遊戲按鍵說明：

通用操作：
--------------------
Q         : 返回上一個介面
            (遊戲中 -> 角色選擇畫面)
            (角色選擇畫面 -> 開始選單)
            (開始選單 -> 退出遊戲)
方向鍵 ←/→ : (角色選擇畫面) 切換角色
Enter       : (角色選擇畫面) 確認選擇的角色


開始選單：
--------------------
空白鍵     : 進入角色選擇畫面


遊戲進行中：
--------------------
空白鍵     : - 如果顯示 "Ready!": 開始遊戲
            - 如果遊戲正在進行: 暫停 / 繼續遊戲
方向鍵 ↑↓←→ : 控制 Pacman 移動
J         : (特定角色) 使用主要技能 (例如：Gunner 射擊)


祝您遊戲愉快！
